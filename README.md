# Django Capstone Project

## Overview
This is a Django capstone project demonstrating full-stack development and software engineering practices. The project includes version control, documentation, and containerisation.

It showcases:
- Django web framework
- Git version control with branching
- Sphinx documentation generation
- Docker containerisation
- Clean, modular project structure

---

## Project Structure

- config/ → Django project settings and configuration  
- store/ → Main application (views, models, urls)  
- templates/ → HTML templates  
- static/ → CSS and static files  
- docs/ → Sphinx documentation  
- requirements.txt → Python dependencies  
- Dockerfile → Container configuration  
- capstone.txt → Submission link file
